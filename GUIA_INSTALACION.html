Portal de Datos — Guía de Instalación en PC Personal
Antes de empezar: Este proceso se hace una sola vez en tu PC. Después todo es automático. Tiempo estimado: 10–15 minutos.

Paso 1 — Instalar Python

1
Descargar Python
Ir a https://www.python.org/downloads/ y descargar la versión más reciente (botón amarillo grande).
2
Instalar — ¡con una opción clave!
Ejecutar el instalador. En la primera pantalla, tildar esta opción antes de continuar:
☑ Add Python to PATH
Si no tildás esta opción, nada funcionará. Es lo más importante.
3
Verificar instalación
Abrir CMD (buscar "cmd" en el menú inicio) y escribir:
python --version
Debería aparecer algo como: Python 3.12.4 — significa que funciona.

Paso 2 — Configurar la carpeta de OneDrive

4
Encontrar tu usuario de Windows
Abrí CMD y escribí:
echo %USERNAME%
Anotá el nombre que aparece (ej: Juan o jperez).
5
Abrir config.ini
En la carpeta del actualizador, hacer click derecho en config.ini → Abrir con → Bloc de notas. Buscar la línea que dice carpeta = y cambiarla con tu usuario:
carpeta = C:\Users\TU_USUARIO\OneDrive\PEGSA_Portal\datos
Reemplazá TU_USUARIO con el nombre que anotaste arriba. Guardar con Ctrl+S.
La carpeta PEGSA_Portal\datos se crea sola, no tenés que crearla a mano.

Paso 3 — Instalar y probar

6
Ejecutar el instalador
Click derecho en 1_INSTALAR.bat → Ejecutar como administrador.
Seguir las instrucciones en pantalla. Tarda 1–2 minutos.
7
Conectar la VPN
Conectar la VPN de la empresa como siempre. Luego verificar que el servidor sea visible:
ping DARWASH
Si aparecen 4 líneas con tiempos (ej: tiempo=5ms), la conexión funciona.
Si dice "Host no encontrado", la VPN no está conectada o hay un problema de red.
8
Primera prueba
Doble clic en 2_EJECUTAR_AHORA.bat.
Va a aparecer una ventana mostrando el progreso. Al final debería decir:
✓ EXITO: Datos actualizados correctamente
Los archivos JSON van a aparecer en tu carpeta de OneDrive y se sincronizan automáticamente.

Uso diario — No tenés que hacer nada

La tarea queda programada para correr todos los días a las 7:00 AM automáticamente, siempre que tu PC esté encendida y la VPN conectada.

Si querés actualizar en cualquier momento: doble clic en 2_EJECUTAR_AHORA.bat.
